# 🚀 ProductiveHQ

![HTML](https://img.shields.io/badge/HTML-5-orange)
![CSS](https://img.shields.io/badge/CSS-3-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)
![Responsive](https://img.shields.io/badge/Responsive-Yes-success)

ProductiveHQ is a modern productivity dashboard built with **HTML, CSS, and JavaScript**. It helps users organize tasks, plan their day, track goals, stay focused with a Pomodoro timer, check the weather, and get inspired with daily motivational quotes.

## 🌐 Live Demo

🔗 **Live:** https://your-live-demo-link.com

## ✨ Features

- 📊 Dashboard Overview
- ✅ Todo Manager
- 📅 Daily Planner
- 🎯 Daily Goals
- 🍅 Pomodoro Timer
- 🌤 Live Weather
- 💡 Daily Quotes
- 🌙 Dark & Light Theme
- 💾 Local Storage Support

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
- Remix Icons

## 🚀 Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.
3. Add your OpenWeatherMap API key in `script.js` to enable the weather feature.
